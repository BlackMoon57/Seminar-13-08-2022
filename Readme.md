## Создание коммитов

### Создание фиксации
Для создания фиксации, используется *gitcommit*. Для этого в терминале с папкой-репозиторием необходимо написать команду *git commit -m <сообщение коммиту>*. Сообщение коммиту писать **ОБЯЗАТЕЛЬНО**.
